Welcome to the Akari Game documentation! This page provides information about the Akari game, a single-\/player logic puzzle implemented in Java using the model-\/view-\/controller (MVC) design pattern with the Java\+FX UI library.

The source code is hosted on Git\+Hub at \href{https://github.com/SamuelToledoHdez/Akari-DAP}{\texttt{ Akari Game Repo}}.

The main code resides in the directory \href{https://github.com/SamuelToledoHdez/Akari-DAP/tree/main/src/main/java/com/comp301/a09akari}{\texttt{ src/main/java/com/comp301/a09akari}}

The tests resides in the directory \href{https://github.com/SamuelToledoHdez/Akari-DAP/tree/main/src/test/java/com/comp301/a09akari}{\texttt{ src/test/java/com/comp301/a09akari}}.

The project is supported by Maven, and continuous integration is performed with Jenkins.\hypertarget{index_application_sec}{}\doxysection{\texorpdfstring{Running the Application}{Running the Application}}\label{index_application_sec}
To run the Akari game application, ensure Maven is installed. Explore the features and functionalities through the user interface.
