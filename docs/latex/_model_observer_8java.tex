\doxysubsection{Detailed Description}
Interface for observing changes in a puzzle model. 

This interface defines a method that should be implemented by classes interested in observing changes in a puzzle model. When a model value is changed, the {\ttfamily update()} method is called on all active Model\+Observer objects.
